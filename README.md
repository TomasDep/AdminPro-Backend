
# AdminPro - Backend

## Descripción Proyecto
AdminPro es un modulo para la administración de una organanización.
El Proyecto es desarrollado con el framework de desarrollo web MEAN (Mongo, Express.js, Angular y Node.js)
## Tecnologías Backend
- Node.js
- Express.js
- Mongoose 
- JWT
- Bcrypt.js
- Google Auth
- UUID
